Pre_replace

2023-12-29 3 min read 
function complex($re, $str) {
    return preg_replace(
        '/(' . $re . ')/ei',
        'strtolower("\\1")',
        $str
    );
}

这是一个名为 complex() 的函数，它使用正则表达式替换来处理字符串。

这个函数接受两个参数：

  • $re：表示正则表达式的模式。
  • $str：表示要进行替换操作的字符串。

函数使用 preg_replace() 函数来执行替换操作。在替换过程中，使用了 e 修饰符，它允许将替换操作视为可执行的代码。

具体而言，在给定的字符串 $str 中，所有匹配正则表达式 $re 的部分都会被替换为对 strtolower("\\1") 的调用。其中 \\1 表示捕获到的匹配结果，并将其转换为小写字母。

需要注意的是，使用 e 修饰符在较新版本的 PHP 中已经被弃用，并且在更高的安全级别下可能会导致安全风险。建议使用其他方式来执行替换操作，例如使用匿名函数或者先把字符串进行拆分和处理，然后再进行替换。

以下是一个改进的版本，使用匿名函数来执行替换操作：

function complex($re, $str) {
    return preg_replace_callback(
        '/' . $re . '/',
        function ($matches) {
            return strtolower($matches[0]);
        },
        $str
    );
}

在这个改进的版本中，我们使用 preg_replace_callback() 函数，它接受一个匿名函数作为替换操作的回调函数。在匿名函数中，我们直接将匹配到的内容转换为小写字母，并返回替换后的结果。

这种改进的方式更加安全且不依赖于 eval 执行代码。

e模式下的preg_replace可以让第二个参数'替换字符串'当作代码执行，但是这里第二个参数是不可变的，但因为有这种特殊的情况，正则表达式模式或部分模式两边添加圆括号会将相关匹配存储到一个临时缓存区，并且从1开始排序，而strtolower("\1")正好表达的就是匹配区的第一个（\1=\1），从而我们如果匹配可以，则可以将函数实现。
比如我们传入 ?.*={${phpinfo()}}

原句：preg_replace('/(' . $re . ')/ei','strtolower("\\1")',$str); 就变成preg_replace('/(' .* ')/ei','strtolower("\\1")',{${phpinfo()}});

又因为$_GET传入首字母是非法字符时候会把 .（点号）改成下划线，因此得将.换成\s
